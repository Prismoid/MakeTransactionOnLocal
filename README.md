# Caution!
using web3@0.16.0

## usage
1. running a Ethereum (or Ethereum Classic) node
2. compile and migrate contracts
```
truffle compile
truffle migrate
```
or
on the truffle console
```
> migrate --reset
```
3. Setting npm init
'''
cd ./js-rpc
npm init
npm install 
'''
4. calling web3.js RPC
```
cd ./js-rpc
node Test.js
```
